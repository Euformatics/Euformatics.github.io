Introduction

This API provides access to your NGS quality data stored in the omnomicsQ quality control system for NGS sequencing data.

The API is available for testing at https://q.omnomics.com/api/v1.

Currently the service is in beta testing. Details may still change before the public launch.

The API uses HTTP Basic authentication. For testing, you can use the same username and password that you use for the omnomicsQ web application. For real usage, please contact Euformatics for an API key.

All operations produce responses in JSON format.

Note
 Please note that the example responses below have been edited for brevity, and are not actually complete/realistic responses that you would get from the actual API.

Listing QC metrics

Returns the list of all QC metrics.
